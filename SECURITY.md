# Security Policy

Security is an important aspect of the `NovaScript` project.

If you discover a security vulnerability, please report it responsibly.

---

## Reporting a Vulnerability

Please report security issues through GitHub security advisories or by contacting the maintainers directly.

Avoid publicly disclosing vulnerabilities until they have been reviewed.

---

## Scope

Security issues may include:

* Compiler vulnerabilities
* Runtime vulnerabilities
* Dependency vulnerabilities
* Build system vulnerabilities

---

## Responsible Disclosure

When reporting a vulnerability, please provide:

* A clear description of the issue
* Steps to reproduce the problem
* Potential impact
* Suggested mitigation (if known)

---

We appreciate responsible disclosure and will work to address security issues as quickly as possible.
