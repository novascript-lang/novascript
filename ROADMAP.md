# `NovaScript` Roadmap

This document outlines the early development roadmap for `NovaScript`.

The roadmap may evolve as the project grows and the community contributes feedback.

---

## Phase 1 — Language Design

Goals:

* Finalize core language philosophy
* Publish initial language specification
* Define the type system and object model
* Establish module and package structure

Milestone:

`NovaScript` v0.1 specification draft.

---

## Phase 2 — Compiler Prototype

Goals:

* Implement parser
* Implement type checker
* Build initial intermediate representation
* Generate WebAssembly output

Milestone:

Basic compiler capable of compiling simple programs.

---

## Phase 3 — Runtime Prototype

Goals:

* Implement runtime in Zig
* Create task scheduler
* Implement async execution model
* Build WASM loader

Milestone:

Executable runtime environment.

---

## Phase 4 — Toolchain Integration

Goals:

* Implement CLI (`ns`)
* Add build system
* Add test runner
* Add formatter and linting

Milestone:

Integrated developer workflow.

---

## Phase 5 — Ecosystem Foundations

Goals:

* Initial standard library
* Playground environment
* Editor tooling (VSCode extension)

Milestone:

Early developer adoption.

---

## Phase 6 — Stable Release

Goals:

* Language stability
* Performance optimization
* Documentation maturity

Milestone:

`NovaScript` v1.0.
