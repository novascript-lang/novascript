# Code of Conduct

The `NovaScript` community is committed to creating a welcoming and respectful environment for all participants.

We expect all contributors to behave with professionalism and respect.

---

## Our Standards

Examples of positive behavior include:

* Being respectful and considerate
* Providing constructive feedback
* Helping new contributors learn
* Encouraging open discussion

Examples of unacceptable behavior include:

* Harassment or discrimination
* Personal attacks
* Trolling or disruptive behavior

---

## Enforcement

Project maintainers are responsible for enforcing this Code of Conduct.

If you experience or witness unacceptable behavior, please report it through the repository discussions or contact maintainers directly.

---

## Scope

This Code of Conduct applies to all project spaces, including:

* GitHub issues
* Pull requests
* Discussions
* Community channels

---

The goal of `NovaScript` is to build a collaborative and respectful open-source community.
